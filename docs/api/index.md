# API Reference

The REST API is fully documented via the OpenAPI specification. The committed snapshot lives at [`docs/api/openapi.json`](https://github.com/derekwinters/chores-web/blob/main/docs/api/openapi.json).

To explore interactively, import `docs/api/openapi.json` into any OpenAPI-compatible tool (Swagger UI, Postman, Insomnia, etc.).
